### 대여소 상세 정보 조회 및 대여, 예약 대기
| Actor Action                 | System Response | 문자 알림 서비스 (외부 액터) |

| | 1. 검색된 대여소 리스트 화면 출력 | |

| 4. 회원은 자전거가 남아 있는 경우 특정 자전거의 대여 요청 | 5. 대여되었다는 메세지 출력 | 6. 대여되었다는 문자 알림을 전송하는 외부 서비스 연동 |
| 7. 회원은 자전거가 없는 경우 특정 자전거의 예약대기 요청 | 8. 예약대기가 되었다는 메세지 출력 | 9. 예약대기가 되었다는 문자 알림을 전송하는 외부 서비스 연동 |
